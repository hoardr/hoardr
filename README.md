# Hoardr
A self-hosted application for tracking your home inventory.

# Installation
> Hoardr is still under development and is not yet ready to be used in your home setup.
> 
> However, you're welcome to test it out locally.

# Testing locally

1. `docker-compose up`
2. `./gradlew bootRun`
3. `cd frontend`
4. `npm run start`
